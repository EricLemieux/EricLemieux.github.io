About

Games

Falling Fat Cat (Working Title)

January 2014

iOS

Swift, SpriteKit

This game was developed while working at relish interactive during the summer of 2014. It was developed for iOS using the new language Swift and SpriteKit.

The gameplay is a mixture of "Bejeweled" and "Falldown", where the user needs to keep the cat from hitting the top of the screen by matching blocks into groups. The gameplay is enhanced with special blocks that have different effects which can make the game easier or harder for the player, depending on the effect.

Rainbow Unicorn Farts

January 2014

PC

C++, OpenGL, SFML

Rainbow Unicorn Farts is a simple 2D endless runner created during a 24 hour game jam. The game focuses on a unicorn trying to stay on a rainbow path.

Developed as part of a 24 hour game jam, under the theme "Space the Untamed Beast". The game was a test for a custom framework that was designed for easily loading 2D art assets.

Robopocalypse

April 2014

PC

C++, OpenGL, GLFW

Robopocalypse is a two person fighting game for the PC, set in the future where battle robots the size of buildings battle for earths enjoyment.

Developed as a culminating project for a university class. I was the lead programmer on this project, I created the engine including the graphics system, file input and output, managing other programmers on the team.
